## 1. PRESENCE

**The digital space and visual identity of the client.**

### What It Generates
- Websites, web apps, dashboards
- Visual brand presence across digital channels
- User interfaces and experiences

### What It Contains
| Technology | Purpose |
|------------|---------|
| Vite + Vanilla JS | Default frontend build |
| SvelteKit / Astro | Complex projects |
| Firebase Auth | User authentication |
| Cloudflare Pages | Hosting & CDN |

### Connections
```
PRESENCE ──▶ INTELLIGENCE   (Smart chatbots, AI features embedded in sites)
PRESENCE ──▶ CONNECTION     (Lead capture forms, contact routing)
```

---

## 2. INTELLIGENCE

**The autonomous thinking layer and logic processing.**

### What It Generates
- AI assistants and chatbots
- Automated decision-making
- Data processing and analysis

### What It Contains
| Technology | Purpose |
|------------|---------|
| LMStudio | Local AI models (Llama) |
| N8N Workflows | Automation engine |
| Firebase Functions | Serverless compute |
| Custom AI Workflows | Chained autonomous operations |

### Connections
```
INTELLIGENCE ──▶ PRESENCE    (AI features power smart websites)
INTELLIGENCE ──▶ CONTINUITY  (Autonomous chained workflows)
```

---

## 3. CONNECTION

**The pipeline linking entities: leads, clients, and data.**

### What It Generates
- Lead generation systems
- Client pipelines and funnels
- Data flow between services

### What It Contains
| Technology | Purpose |
|------------|---------|
| N8N Workflows | Lead routing automation |
| Firebase Firestore | Client data storage |
| Google Ecosystem | Email, calendar, contacts |
| Smart Forms | Data capture & validation |

### Connections
```
CONNECTION ──▶ PRESENCE     (Lead capture feeds into sites)
CONNECTION ──▶ CONTINUITY   (Automated CRM, client relationships)
```

---

## 4. CONTINUITY

**The force that enables system perpetuation without manual intervention.**

### What It Generates
- Self-sustaining operations
- Health monitoring
- Automated maintenance and scaling

### What It Contains
| Technology | Purpose |
|------------|---------|
| Uptime Scout | Health monitoring |
| Synthetic Testing | Proactive issue detection |
| Google Automations | Ecosystem continuity |
| SEO Systems | Long-term discoverability |

### Connections
```
CONTINUITY ──▶ INTELLIGENCE  (Workflows run autonomously)
CONTINUITY ──▶ CONNECTION    (CRM operates without manual input)
```

---

## The Interconnection Logic

Opposite forces never touch directly — they communicate through wait-points and feedback loops.

| Adjacent Pair | Interaction |
|---------------|-------------|
| PRESENCE ↔ INTELLIGENCE | Smart websites with AI features |
| PRESENCE ↔ CONNECTION | Lead capture and routing |
| INTELLIGENCE ↔ CONTINUITY | Autonomous chained workflows |
| CONNECTION ↔ CONTINUITY | Automated client relationship management |
